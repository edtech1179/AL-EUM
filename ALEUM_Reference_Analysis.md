## 1. 참고 레포 분석 요약

### 1-1. neisapi-ng (NEIS API 라이브러리)

| 항목 | 내용 |
|------|------|
| **기술** | Python, requests/aiohttp, async 지원 |
| **NEIS 엔드포인트** | schoolInfo, mealServiceDietInfo, SchoolSchedule, classInfo, 시간표 |
| **직접 재사용** | ✅ 바로 사용 가능 — 급식, 학사일정, 학교정보 |
| **부족한 점** | 다국어 변환 없음, 학생 맥락 없음 |
| **알:음 차별점** | 다국어 래퍼 + 학생 맥락(중퇴 위험, 언어 선호도) 추가 |

### 1-2. VirtualQuizGame-OpenCV (핸드트래킹 퀴즈)

| 항목 | 내용 |
|------|------|
| **기술** | OpenCV, cvzone, MediaPipe, CSV 기반 문제 |
| **제스처** | 검지+중지 거리 ≤30px → 선택. 4개 답 영역 매핑 |
| **직접 재사용** | △ 아키텍처만 — Python 데스크탑 → 웹(TensorFlow.js) 포팅 필요 |
| **알:음 적용** | 교육 용어 플래시카드 + O/X 퀴즈 + 4지선다로 변환 |
| **알:음 차별점** | 웹 브라우저 실행(설치 불필요) + 교육 용어 특화 + 난이도 AI 자동 조절 |

### 1-3. lalah-chatbot (다국어 교육 챗봇 — 아프간 학생용)

| 항목 | 내용 |
|------|------|
| **기술** | LangChain, OpenAI/Gemini, FastAPI, Redis, ChromaDB (RAG) |
| **다국어** | 다리어, 파슈토어, 영어 자동 감지 |
| **교육 기능** | 읽기 이해(적응형), 시험 준비, 작문 평가, 과목 튜터링 |
| **배포** | WhatsApp, Telegram, Web |
| **알:음 배울 점** | Multi-LLM 지원, RAG + 교육 용어집, 문화 민감성 프레임워크 |
| **알:음 차별점** | Gemini 네이티브(비용 절감), NEIS 실시간 연동(lalah는 수동), 한국 교육 특화 |

### 1-4. AI_Halal_Product_Detector (할랄 식품 감지)

| 항목 | 내용 |
|------|------|
| **기술** | FastAPI + Next.js, YOLOv8(로고 감지), Tesseract OCR, Supabase |
| **감지 방식** | OCR → 키워드 매칭 → YOLOv8 로고 → DB 조회 → 신뢰도 합산 |
| **직접 재사용** | ✅ 결정 엔진 패턴 (다중 신호 투표 방식) |
| **알:음 적용** | 급식 메뉴 텍스트에서 할랄 키워드 + 알레르기 코드 자동 감지 |
| **알:음 차별점** | NEIS 급식 API에서 알레르기 코드(1~19) 이미 제공 → OCR 불필요, 키워드 매칭만으로 충분 |

### 1-5. dropout-prediction (학생 중퇴 예측 ML)

| 항목 | 내용 |
|------|------|
| **알고리즘** | Decision Tree(83%), Random Forest, SVM, Logistic Regression, KNN |
| **주요 Feature** | 입학 연령, 학비 납부, 1/2학기 성적, 부모 학력/직업 |
| **데이터** | 10년간 대학 데이터, 34개 변수 |
| **알:음 적용** | sklearn 파이프라인 + feature engineering 참고 |
| **알:음 차별점** | 다문화 특화 변수 추가 (한국어 수준, 알림 확인률, MEGI 지수, 학생 유형) |

### 1-6. Multi-language-Course-Content-Translator-Agent (RAG 번역)

| 항목 | 내용 |
|------|------|
| **기술** | IBM Watsonx.ai, LangGraph, Granite 모델, 용어집 RAG |
| **RAG 구조** | 용어집 업로드 → 문서 검색 → LLM이 용어집 참고 번역 |
| **용어집** | 정규식 매칭으로 기술 용어 → 환각 방지 |
| **알:음 배울 점** | 용어집 기반 번역이 환각 방지에 효과적 |
| **알:음 차별점** | Gemini(비용 절감) + 한국 교육 용어 250개 × 6개 언어 + NEIS 실시간 연동 |
