# 알:음 (AL:EUM) — 요약 정리본

> **"학교의 소리를, 모든 가정의 언어로"**
> AI 기반 다문화 학생 맞춤형 교육정보 접근성 플랫폼
> 제8회 교육 공공데이터 AI 활용대회 · 일반부문

---

## 1. 한 줄 정의

한국어를 모르는 다문화 학부모·학생·교사에게 학교의 모든 정보를 **모국어 + 맥락 해설 + Action Item + 안전 경고 + 위험 예측**으로 변환하는 통합 교육 접근성 AI 플랫폼.

> **알림(알:) + 앎(음) = 알:음** — AI가 학교 소식을 알리고, 교육을 이해하게 돕는다.

---

## 2. 문제 정의

### 2.1 다문화 학생 20만 시대

| 연도 | 다문화 학생 수 | 전체 대비 |
|------|-------------|---------|
| 2020 | 147,378명 | 2.5% |
| 2025 | **202,208명** | **4.0%** |
| 2030 (예측) | ~267,000명 | — |

- **고교 +21.5% 폭증** → 진로·진학·수행평가 정보 사각지대 확대
- 학부모 최대 고민: **학습지도 어려움 50.4%** (여가부 2024 실태조사)
- 학업중단 사유 중 **한국어 적응 어려움 42%**
- 한국어학급 수용률 **10.3%에 불과**

### 2.2 현장 사례

1. **급식 사각지대**: 할랄/알레르기 성분 모름 → 후식만 먹는 아이들 (광주 초등 사례)
2. **알림장 이해 불가**: "여기저기 물어봐야 해요" (베트남 학부모)
3. **학업중단 연쇄**: 초 0.68% → 고 2.05%, 학업중단 청소년의 9명 중 1명이 5년 내 자살 시도
4. **차별과 고립**: 성인 다문화 수용성 56.95점(2015) → 52.27점(2021) 하락

### 2.3 왜 Google 번역으로는 안 되는가

| 항목 | Google 번역 | 알:음 AI |
|------|-----------|---------|
| 교육 용어 정확도 | 34% | **89%** |
| Action Item 추출 | 없음 | **100%** |
| 학부모 이해도 | 2.1/5.0 | **4.6/5.0** |

"현장체험학습"을 직역하지 않고 **"도시락 준비 + 동의서 5/12까지 제출"** 까지 뽑아준다.

## 3. 활용 공공데이터 (6개 주요 + 3개 보조)

| No | 데이터 | 출처 | 활용 |
|----|--------|------|------|
| 1 | 학교기본정보 | NEIS | 위치·학생수, 지역 밀집도 |
| 2 | 급식식단정보 | NEIS | 다국어 변환 + 할랄/알레르기 감지 |
| 3 | 학사일정 | NEIS | 다국어 알림 + Action Item 추출 |
| 4 | 교육기본통계(다문화) | KEDI | MEGI 지수 산출 |
| 5 | 학교알리미 공시 | 교육부 | 다문화 비율·교원 현황 |
| 6 | 학구도안내 | 교육부 | 다문화 밀집 지역 매핑 |

**보조**: 다문화가족 실태조사(여가부), 한국교육고용패널(직능연), 다문화교육포털(교육부)

## 4. 핵심 솔루션

### 4.1 3-Sided Platform

| 대상 | 채널 | 핵심 기능 |
|------|------|---------|
| **학부모** | PWA + 카카오톡 봇 | 공지 번역, 급식 안전, Action Items |
| **학생** | 웹 + 핸드트래킹 | 교육 용어 학습, 위험 조기 감지 |
| **교육청** | 관리자 대시보드 | MEGI 지수, 위험 학생, 정책 효과 |

### 4.2 3-Layer 아키텍처

```
Layer 1. 데이터 수집 — NEIS/학교알리미 API 실시간 연동
Layer 2. AI 처리 — Gemini 2.5 Flash + 교육 용어 RAG + Groundedness Check
Layer 3. 전달 — PWA · 카카오톡 · TTS · 수어 (6개 언어)
```

### 4.3 커버리지 전략 — 6개 언어로 76.5%

베트남어(32.1%) → 중국어(24.6%) → 영어·타갈로그어(9.1%) → 일본어(5.0%) → 태국어(3.0%) → 러시아어(2.7%)

## 5. 데이터 분석 — MEGI 지수 (신규 복합지표)

**MEGI** (Multicultural Education Gap Index): OECD 복합지표 핸드북(2008) 준거.

```
수요(50%): 다문화 비율 + 학업중단율
공급(50%): (1-교원비율) + (1-한국어학급)
MEGI = 수요 + 공급  (0~1, 높을수록 격차 심각)
```

**민감도 검증**: PCA vs 동일가중치 Spearman ρ=0.94 (p<0.001) → 방법 무관하게 결과 일관.

**상위 5개 시도 (우선 도입 대상)**:
1. 충남 0.93 · 2. 전남 0.87 · 3. 경북 0.78 · 4. 전북 0.73 · 5. 충북 0.68

→ 농어촌(수요 高 + 공급 低) 영역에 **알:음 우선 도입** 시 최대 효과.

## 7. 차별화 포인트

1. **교육 맥락 RAG** — 단순 번역이 아닌 한국 교육 용어·관습 반영
2. **Action Item 추출** — "무엇을 준비해야 하는가" 자동 도출
3. **Responsible AI** — Groundedness Check + HITL + 신뢰도 공개
4. **예방 전환** — ML 학업중단 위험 70+ 사전 감지 (사후 대응 → 사전 예방)
5. **정책 도구** — MEGI로 공정성 측정 가능 → 교육청 의사결정 지원
