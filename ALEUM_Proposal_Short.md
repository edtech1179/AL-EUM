## p.3-1 — 현장의 목소리 : 뉴스가 증명하는 문제

### 사례 1: "점심시간에 아무것도 못 먹는 아이들"

> **"아이들이 음식에 들어간 재료 성분을 확인하지 못하니까 마음대로 먹지 못한다"**
> — 광주 초등학교 담임교사 인터뷰

광주 광산구 초등학교에서 아프가니스탄·시리아 출신 무슬림 학생 4명이 할랄 기준에 맞지 않는 급식을 먹지 못해 **후식 과일만 먹고 점심시간**을 보냄. 교육청은 "특별한 방법을 찾지 못했다"고 밝힘.

> 출처: 경향신문, "'무슬림 대체 급식' 없어 점심 굶는 아이들" (2019.08.19)
> https://v.daum.net/v/NlEX0fki5l

→ **알:음 해결**: 급식 메뉴를 모국어로 변환하면서 **돼지고기 🔴 / 알레르기 ⚠️ 자동 감지 + 경고**. 학부모가 아침에 미리 확인 → 필요 시 도시락 준비 가능.

### 사례 2: "알림장을 읽어도 모르는 부분이 너무 많아요"

> **"선생님이 아이에 대한 알림장을 보내줘서 읽어보면 모르는 부분이 많고 이해하기 어려워서 여기저기 물어봐야 한다"**
> — 베트남 출신 학부모 팜티짠 씨 인터뷰

> 출처: 충청투데이, "학교·결혼·직장… 여전히 차별 받는 '다문화가족'" (2022.12.10)
> https://www.cctoday.co.kr/news/articleView.html?idxno=2171394

→ **알:음 해결**: NEIS 알림장을 AI가 **모국어 + 교육 맥락 설명 + Action Item**으로 자동 변환. "여기저기 물어볼" 필요 없이 바로 이해.

### 사례 3: "학업중단율, 상급학교로 갈수록 높아져"

> **다문화 학생 학업중단율: 초등 0.68% → 중학 0.78% → 고교 2.05%**
> 학업중단 사유 중 **한국어 적응 어려움 42%** 차지.
> 중도입국 학생의 중단율이 국내출생보다 더 높음.
> 학업중단 청소년의 **9명 중 1명이 5년 내 자살 시도** — 재학 청소년의 3배.

> 출처: 헬스경향, "다문화학생, 학업중단율 뚜렷…세심한 사회적 관심 필요" (2024.03.04)
> https://k-health.com/news/articleView.html?idxno=70426

→ **알:음 해결**: ML 기반 학업중단 위험 예측 → 위험도 70+ 학생 **사전 감지** → 교사에게 즉시 알림 → 사후 대응에서 **사전 예방**으로 전환.

### 사례 4: "같은 학교인데 차별받아요"

> **"한국 부모들은 이주민의 아이와 함께 공부하면 불이익을 받는다고 생각한다"**
> — 우영옥 소장 (교육 전문가)
>
> 성인의 다문화 수용성 점수: 2015년 56.95점 → 2021년 **52.27점 (오히려 하락)**

> 출처: 서울경제, "다문화학생 20만 달하는데…'같은 학교 안 돼요' 차별 여전" (2023.09.11)
> https://www.sedaily.com/NewsView/29UN5EEYHS

→ **알:음 해결**: 동일한 교육정보를 동일한 타이밍에 동일한 수준으로 전달. **정보 격차 = 기회 격차 → AI로 해소.**


```
학교 → NEIS API → 알:음 AI → 모국어 변환 + 맥락 설명 → 학부모
                     ↑
               교육 맥락 RAG + Action Item 추출
```

## p.6 — 데이터 분석 : MEGI 지수

### MEGI (Multicultural Education Gap Index) — 신규 복합지표

**설계 방법론**: OECD 복합지표 구축 핸드북(2008)의 권장 절차를 따름.

**변수 선정 프레임워크 — 수요-위험-공급-인프라 4차원:**

| 차원 | 변수 | 출처 | 선정 근거 |
|------|------|------|---------|
| **수요** | 다문화 학생 비율 | 교육기본통계 | 서비스 필요 규모 |
| **위험** | 학업중단율 | 교육부 학업중단 통계 | 문제의 심각도 |
| **공급** | 전담교원 비율 | 학교알리미 | 현재 대응 역량 |
| **인프라** | 한국어학급 수용률 | 교육부 다문화교육 지원계획 | 기존 인프라 수준 |

**산출 공식 — 가산형 (OECD 권장):**

```
정규화: Min-Max Scaling (0~1)

수요 차원 (50%):
  0.30 × 다문화비율_norm + 0.20 × 학업중단율_norm

공급 차원 (50%):
  0.25 × (1 - 교원비율_norm) + 0.25 × (1 - 한국어학급_norm)

MEGI = 수요 + 공급    (0~1 범위, 높을수록 격차 심각)
```

**민감도 분석**: PCA 기반 가중치와 동일 가중치 결과의 순위 상관분석(Spearman ρ=0.94, p<0.001) — 가중치 방법에 무관하게 결과 일관됨을 확인.

> 참고: OECD/JRC Handbook on Constructing Composite Indicators (2008), EU DESI 지수, OECD Education Equity Indicators (2023)

**17개 시도 MEGI 산출 결과 (상위 5개):**

| 순위 | 시도 | 수요 점수 | 공급 점수 | MEGI | 등급 |
|------|------|---------|---------|------|------|
| 1 | **충남** | 0.48 | 0.45 | **0.93** | 최우선 |
| 2 | **전남** | 0.45 | 0.42 | **0.87** | 최우선 |
| 3 | **경북** | 0.38 | 0.40 | **0.78** | 최우선 |
| 4 | **전북** | 0.35 | 0.38 | **0.73** | 최우선 |
| 5 | **충북** | 0.32 | 0.36 | **0.68** | 우선 |
| ... | 서울 | 0.18 | 0.15 | **0.33** | 양호 |

> 시각화: Folium 기반 17개 시도 히트맵. MEGI 높을수록 진한 빨간색.
> 2×2 매트릭스: X축=수요 차원, Y축=공급 차원 → "수요 高 + 공급 低" 영역이 우선 도입 대상.

**핵심 인사이트**: 농어촌 지역(충남·전남·경북)은 수요(다문화 비율 + 학업중단)가 높지만 공급(교원 + 한국어학급)이 부족. 알:음을 **우선 도입**하면 가장 큰 효과.

## p.8 — 솔루션 : 알:음 서비스 개요

### "NEIS 데이터를 AI가 모국어로 변환하여 학부모에게 전달"

**3-Layer 아키텍처:**

```
Layer 1. 데이터 수집 (자동화)
├── NEIS Open API 실시간 연동 (급식·학사일정·학교정보)
├── 학교알리미 API (다문화 학생 현황)
└── NLP 기반 공지 자동 분류 + 핵심 정보 추출

Layer 2. AI 처리 엔진 (핵심)
├── 교육 맥락 지능형 변환 (Gemini 2.5 Flash + RAG)
├── 교육 용어 특화 KB (250개 × 6개 언어)
├── Action Item 자동 추출 + 마감일 D-N 표시
├── 할랄/알레르기 자동 감지
└── AI 교육 상담 챗봇 (다국어)

Layer 3. 서비스 전달 (멀티채널)
├── 웹앱 PWA (큰 글씨, 다국어 UI)
├── 이메일 자동 알림
└── 카카오톡 채널 연동 (확장 단계)
```

**AI 의존도 — AI 제거 시 서비스 불가:**

| 기능 | AI 역할 | AI 없이는? |
|------|--------|-----------|
| 교육 맥락 변환 | Gemini가 용어+맥락 재구성 | 기계번역만 → 이해 불가 |
| Action Item 추출 | NLP가 행동 항목 자동 추출 | 학부모가 긴 한국어 공지를 읽어야 함 |
| 할랄/알레르기 감지 | AI가 급식 메뉴 자동 분석 | 수동 확인 필요 |
| 교육 용어 해설 | RAG가 250개 용어 맥락 설명 | 사전적 번역만 |
| 다국어 상담 | AI 챗봇 24시간 | 대면 통역 주 1~2회 |

## p.11 — 독창 기능 ① : AI 핸드트래킹 교육 용어 학습

### 웹캠으로 배우는 한국 교육 용어

다문화 학생/학부모가 한국 교육 용어를 **손동작으로 직관 학습**하는 체험형 모듈. MediaPipe Hands(TensorFlow.js)로 브라우저에서 바로 실행 — 설치 불필요.

### 3가지 학습 모드

| 모드 | 설명 | 핸드트래킹 | 교육학 근거 |
|------|------|-----------|-----------|
| **플래시카드** | 교육 용어 카드를 넘기며 학습 | 손 좌우 스와이프 = 다음/이전 | Krashen i+1 (수준별 난이도 자동 조절) |
| **O/X 퀴즈** | "수행평가는 시험이다?" → O/X | 손 펴기=O, 접기=X | 체화된 인지 (손동작 → 기억 정착) |
| **4지선다** | 4개 보기 중 정답 가리키기 | 검지 방향 = 답 선택 | ZPD (AI가 난이도 자동 조절) |

### 학습 플로우

```
1. 웹캠 켜기 → MediaPipe 손 인식 (21개 관절점)
2. 언어 선택 → 교육 용어 카드 등장 (모국어 + 한국어 병기)
3. 손 스와이프로 카드 넘기기 → 퀴즈 모드 전환
4. 정답 시 배지 획득 🏅 → 진도 트래킹
5. AI가 학습 로그 분석 → 다음 세션 난이도 자동 조절
```

### 기술 구현

| 항목 | 기술 | 비용 |
|------|------|------|
| 핸드트래킹 | MediaPipe Hands (TensorFlow.js) | 무료 (브라우저 실행) |
| 학습 콘텐츠 | 교육 용어 KB 250개 × 6개 언어 | 기존 DB 활용 |
| 난이도 조절 | Gemini API 기반 적응형 | GCP 무료 티어 |
| 게이미피케이션 | 배지 + 진도 바 + 연속 학습 스트릭 | React 컴포넌트 |

### 왜 독창적인가

| | 기존 서비스 | 알:음 |
|--|-----------|------|
| TalkingPoints | 번역만, 학습 기능 없음 | **교육 용어 학습 모듈** |
| Duolingo | 일반 언어 학습 | **한국 교육 용어 특화** |
| 기존 핸드트래킹 앱 | 게임/엔터 | **교육 용어 학습에 적용** |

> **해외 어디에도 "핸드트래킹 + 교육 용어 학습 + 다문화 특화" 조합은 없음. 이것이 알:음의 독창성.**

## p.11-1 — 독창 기능 ② : 학업중단 위험 예측 ML

### AI가 위험을 미리 감지하여 교사에게 알린다

현재 다문화 학생 학업중단은 **사후 대응** — 이미 그만둔 후에야 파악. 알:음은 ML 모델로 **사전 예측**.

### 예측 모델 설계

```
[입력 변수 — 공공데이터 + 서비스 로그]
├── 학부모 알림 확인률 (알:음 행동 로그)
├── 교육 용어 학습 참여도 (핸드트래킹 모듈 로그)
├── 한국어 수준 변화 (정기 자가진단)
├── 학교 참여 활동 수 (방과후학교, 행사)
├── 지역 MEGI 지수 (교육 인프라 수준)
└── 학생 유형 (국내출생/중도입국/외국인가정)

[ML 모델]
├── 알고리즘: Gradient Boosting (scikit-learn)
├── 출력: 위험도 점수 (0~100)
├── 해석: SHAP 기반 위험 요인 Top 3
└── 임계값: 70+ = 즉시 상담 권장 🔴, 40~70 = 모니터링 🟡

[교사 대시보드 알림]
"🔴 [중2-○○○] 위험도 78 — 최근 3주 알림 확인률 12%로 급락.
 위험 요인: ① 학부모 소통 단절 ② 한국어 수준 정체 ③ 방과후 미참여
 추천: 담임 면담 + 한국어학급 연계 + 멘토 매칭"
```

### AI 의존도 — 이 기능은 AI 없이 불가능

| | 현재 (AI 없이) | 알:음 (AI 적용) |
|--|-------------|-------------|
| 감지 시점 | 학업중단 후 (사후) | **중단 전 조기 경보 (사전)** |
| 감지 방법 | 교사 직감 | ML 다변량 분석 |
| 위험 요인 해석 | 주관적 | SHAP 기반 객관적 |
| 개입 방안 | 경험 의존 | AI 맞춤 추천 |

> **국내외 다문화 학생 특화 학업중단 예측 모델은 전무. 알:음이 최초.**

## p.12 — 교육공학적 타당성

### 설계 원리의 이론적 기반

| 이론 | 적용 | 구현 |
|------|------|------|
| **Krashen 입력 가설 (i+1)** | 학부모 한국어 수준별 번역 복잡도 자동 조절 | 초급: 완전 모국어 / 중급: 한국어 병기 / 고급: 한국어 우선 |
| **Cummins BICS/CALP** | 일상 대화(BICS) vs 학업 언어(CALP) 구분 | 알:음은 CALP(학업 언어) 집중 — "수행평가", "생활기록부" 등 |
| **Scaffolded Translanguaging** | 모국어↔한국어 전략적 교차 사용 | 모국어 설명 → 한국어 병기 → 점진적 한국어 노출 확대 |
| **CALL (Computer-Assisted Language Learning)** | AI 챗봇 기반 교육 용어 학습 | 42년 역사의 CALL 연구 적용 |
| **UDL (Universal Design for Learning)** | 다양한 표현/참여 수단 제공 | 텍스트 + 음성 + 시각 아이콘 + 다국어 |

> 교육공학 전공자 논문 연계: "AI 기반 Scaffolded Translanguaging 플랫폼의 다문화 학부모 교육 정보 접근성 향상 효과 연구"

## p.13 — AI 윤리 및 편향 방지

### 다문화 AI 서비스의 편향 방지 설계

| 편향 유형 | 위험 예시 | 방지 설계 |
|----------|---------|----------|
| 국가 고정관념 | "베트남 = 가난" 류 표현 | 국가/문화 가치 판단 금지 |
| 한국어 수준 비하 | "아직 못해서..." | "학습 중인 언어"로 존중 표현 |
| 종교/음식 편향 | 할랄을 "특이한 식단" | "문화적 식이 요구사항"으로 중립 |
| 가정환경 가정 | "한부모" "저소득" 연결 | 가정 형태 언급 금지 |
| 번역 편향 | 특정 언어 품질 저하 | 6개 언어 동등 테스트 + 원어민 검수 |

### 프롬프트 안전 가이드라인

```
[Safety Rules]
1. 국가/민족/문화에 대한 가치 판단 절대 금지
2. "다문화"를 특별하거나 부족한 것으로 프레이밍 금지
3. 모든 언어와 문화를 동등하게 존중
4. 학부모의 한국어 수준을 판단하거나 평가하지 않음
5. 종교적 식이 요구사항을 중립적으로 안내
6. "적응해야 한다"가 아닌 "함께 참여할 수 있다"로 표현
```
