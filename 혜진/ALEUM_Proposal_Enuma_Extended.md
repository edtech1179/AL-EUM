## [페이지 0] 확장 서문 — 왜 "에누마 렌즈"로 다시 기획했는가

### 에누마(Enuma)가 던지는 질문

2012년 버클리에서 창업한 에누마는 "**전 세계의 모든 아이들이 자기의 속도대로 공부할 수 있는 미래를 만든다**"는 미션 아래, **"학습이 어려운 아이들도 혼자서 공부할 수 있는 최고의 디지털 제품"** 을 만들어 왔다 (출처: enuma.com/ko).

이 미션은 다문화 교육에 결정적 시사점을 준다.

| 에누마의 관찰 | 알:음이 마주한 동일한 진실 |
|---|---|
| 학습 속도가 느린 아이에게 획일적 교재를 주면 낙오한다 | 한국어 수준이 낮은 다문화 학부모에게 한국어 공지만 보내면 정보에서 낙오한다 |
| 교사가 부족한 개발도상국에서도 아이들은 혼자 학습할 수 있어야 한다 | 통역사가 상시 배치될 수 없는 한국 공립학교에서도 다문화 가정은 정보를 이해할 수 있어야 한다 |
| "모두를 위한 교육이 정말 '모두'가 되게 하려면" 소수자에게 맞춰 설계해야 한다 — 김현주(에누마), 2026 WiET in Bloom 강연 | 알:음은 "가장 한국어를 모르는 학부모"를 설계 기준선으로 삼는다 |

### 에누마의 실증 — 2015~2019 Global Learning XPRIZE

에누마는 **일론 머스크가 기부한 1,500만 달러 상금**의 Global Learning XPRIZE에서 공동우승했다. 탄자니아 난민촌의 **교사 부재 환경**에서 태블릿만으로 기초 읽기·쓰기·셈하기 능력 습득이 가능함을 임상적으로 증명한 것이다 (출처: enuma.com/ko). 이는 알:음에 두 가지 이론적 자산을 제공한다.

1. **"교사·통역사 부재 → 학습 불가"는 거짓이다**. 잘 설계된 AI는 이 공백을 메울 수 있다. → 알:음의 **24시간 다국어 챗봇**이 "대면 통역 주 1~2회"를 대체할 수 있다는 주장의 학술적·실증적 근거
2. **"자기 속도대로의 학습"이 효과적이다**. → 알:음의 **학부모 한국어 수준별 비계(scaffolding)** 설계가 "획일적 번역" 대비 우월하다는 근거

## [페이지 3] 문제 정의 — 데이터로 보는 다문화 교육 현실 (확장)

### 3-1. 다문화 학생 20만 시대 (기존 유지)

2025년 교육기본통계: 다문화 학생 **20만 2,208명** (전체의 4.0%), 2012년 대비 4.3배 성장. 2030년 30만 명 돌파 전망.

### 3-2. 학교급별 증가율 — 고교 21.5% 폭증 (기존 유지)

고등학교 +21.5% → 진로·진학·수행평가 복잡 정보 접근 격차 심화.

### 3-3. ⭐ 신규 문제 정의 — "언어 장벽이 만드는 구조적 느린 학습자"

**에누마가 정의하는 "학습이 어려운 아이들"**:
> "완전 문맹의 환경에 놓인 아이들까지 포함하여, 학습이 어려운 아이들도 혼자서 공부할 수 있는 제품을 만든다" — enuma.com/ko

**한국 맥락의 재해석**:
한국에 온 다문화 학생은 **인지 능력이 낮아서가 아니라 언어 장벽 때문에** 수업 이해도가 낮아지고, 교사의 눈에는 "느린 학습자"로 보이게 된다. 이는 지적 능력의 문제가 아니라 **정보 접근성의 문제**다.

**정량 지표**:
| 지표 | 다문화 학생 | 일반 학생 | 격차 |
|---|---|---|---|
| 기초학력 미달률(추정) | 15~20% | 6~8% | **2~3배** |
| 학업중단율 | 1.14% | 0.69% | 1.65배 |
| 고등학교 학업중단율 | 2.05% | 약 1.2% | 1.7배 |
| 학업중단 사유 "한국어 적응" | **42%** | — | — |

> **핵심 재정의**: 알:음이 해결하는 문제는 단순히 "한국어 모르는 학부모의 소통 불편"이 아니다. **"언어 장벽으로 인해 인지적 능력과 무관하게 '느린 학습자' 범주로 밀려나는 구조"** 를 해체하는 것이다. 에누마의 "자기 속도" 철학은 이 구조적 문제의 해결 원리가 된다.

### 3-4. 현장의 목소리 — 뉴스가 증명하는 문제 (기존 유지)

- "점심시간에 아무것도 못 먹는 아이들" (경향신문 2019.08.19)
- "알림장을 읽어도 모르는 부분이 너무 많아요" — 베트남 학부모 (충청투데이 2022.12.10)
- 학업중단 청소년 9명 중 1명이 5년 내 자살 시도 (헬스경향 2024.03.04)
- "같은 학교 안 돼요" 차별 여전 (서울경제 2023.09.11)

### 각 축을 뒷받침하는 연구 (에누마 XPRIZE 기반 보강)

| 축 | 연구 | 핵심 발견 | 알:음 적용 |
|---|---|---|---|
| 가족참여 | Epstein (2001), Henderson & Mapp (2002), Jeynes (2012), TalkingPoints RCT (2023) | 가족 참여 효과크기 d=0.5~0.7 | 다국어 알림 + Action Item |
| **자기주도학습 🆕** | **Global Learning XPRIZE 2015-2019** (enuma.com/ko) | **교사 부재 탄자니아 난민촌에서 15개월간 태블릿 학습만으로 기초 문해·수리 습득. 에누마 Kitkit School 공동우승** | **알:음의 "교사 통역 없이도 학부모·학생이 스스로 학교 정보에 접근"하는 설계 원칙의 실증 근거** |
| **자기주도학습 🆕** | **Zimmerman (2002) — Self-Regulated Learning** | **자기조절 학습의 3단계(계획-수행-평가) 모형** | **알:음 학부모 대시보드의 "이번 주 확인할 것" → "완료 체크" → "피드백" 루프** |
| **자기주도학습 🆕** | **Knowles (1975) — Self-Directed Learning** | **성인 학습자의 자발성을 존중하는 설계 원리** | **다문화 학부모를 "지도받아야 할 대상"이 아닌 "스스로 학습하는 주체"로 위치시키는 UX** |
| 신뢰관계 | Kim & Sheridan (2015), Goodall & Montgomery (2014) | 언어 장벽 제거 시 참여 수준 비다문화와 동등 | 알:음 챗봇 24시간 응답 |

### ⭐ 에누마 3대 설계 원칙을 알:음에 이식

**에누마 원칙 1: 게이미피케이션 (Gamification)**
- 에누마: 토도수학·글방의 탐험 모드, 레벨, 배지, 연속 학습 스트릭
- 알:음 적용: **학부모 주간 알림 완주 배지 + 자녀와 함께하는 "한국어 학습 스트릭" + 핸드트래킹 퀴즈 점수**

**에누마 원칙 2: 적응형 AI (Adaptive AI)**
- 에누마: AI가 학습자의 수준·속도·오답 패턴을 분석해 다음 문제 난이도 자동 조절
- 알:음 적용: **학부모 한국어 수준별 번역 복잡도 자동 조절 (초급 모국어 / 중급 병기 / 고급 한국어 우선) + 정기 자가진단으로 레벨 업데이트**

**에누마 원칙 3: 유니버설 디자인 (Universal Design for Learning, UDL)**
- 에누마: "교실의 다양한 아이들에게 도움을 주기 위해" 설계 — 느린 학습자·특수교육·다문화 공통으로 쓸 수 있는 UI
- 알:음 적용: **학부모 리터러시 수준 무관 사용 가능한 UI (큰 글씨 + 음성 재생 + 아이콘 + 다국어) → WCAG 2.1 AA + UDL 2.2 동시 준수**

## [페이지 4] 데이터 분석 ① — 지역별 다문화 교육 수요-공급 격차 (기존 유지)

### 4-1. 기존 번역 서비스가 실패하는 이유 (기존 유지)

| 서비스 | 한계 | 알:음 해결 |
|---|---|---|
| Google/Papago | 직역 | 교육 맥락 RAG |
| 다문화교육포털 | 정적 문서 | NEIS 실시간 연동 |
| 각 교육청 번역 | 지역 한정 | 전국 + 6개 언어 |
| 대면 통역 | 주 1~2회 | 24시간 AI 챗봇 |

### 4-2. 시도별 다문화 학생 밀집도 (기존 유지)

충남 7.5% · 전남 7.3% · 경북 5.8% · 전북 5.5% · 충북 5.2% · 서울 3.1%

### 4-3. MEGI 지수 (기존 유지)

```
MEGI = (다문화 학생 비율 × 학업중단율) / (전담교원 비율 × 한국어학급 수용률)
```

**17개 시도 결과**: 충남 7.17 · 전남 5.34 · 경북 4.56 · 전북 3.85 · 충북 3.10 · 경기 1.56 · 서울 0.73

### ⭐ 4-4. 신규 지표 — SLI (Slow Learner Intersection Index)

에누마 관점을 반영한 **다문화-느린 학습자 중첩 지수** 신설:

```
SLI(school) = (다문화 학생 비율) × (기초학력 미달률) / (학습 지원 인력 비율)
```

**의미**: SLI가 높은 학교는 "다문화 학생이 많고, 기초학력 미달률이 높고, 학습 지원 인력이 부족한" 곳 — 즉 **에누마의 "자기주도 학습 도구가 가장 절실한" 환경**. 알:음 프로토타입의 **최우선 파일럿 대상**으로 선정.

**시각화 제안**: MEGI × SLI 2×2 매트릭스 → "고 MEGI + 고 SLI" 사분면(우상단)이 도입 우선순위 1번.

## [페이지 6] AI 솔루션 아키텍처 — 6개 모듈 (1개 신설)

### 3-Layer 아키텍처

```
Layer 1. 데이터 수집 (자동화)
├── NEIS Open API 실시간 연동
├── 학교알리미 API

Layer 2. AI 처리 엔진 (6개 모듈)
├── 모듈 1: 교육 맥락 지능형 변환 (LLM + RAG)
├── 모듈 2: 학생 유형 진단 + 맞춤 경로 (ML 분류, 4유형)
├── 모듈 3: 교육 용어 RAG (250개 × 6개 언어)
├── 모듈 4: 다국어 학부모 챗봇
├── 모듈 5: 학업중단 위험도 예측 (ML + SHAP)
└── 🆕 모듈 6: 자기주도 학습 지원 엔진
    ├─ 쉬운 한국어 변환기 (원문 → Grade 3 수준)
    ├─ 에누마 글방 7단계 기반 학습 카드 생성기
    ├─ 학습 스트릭·배지·진도 시스템
    ├─ 음성 재생 (TTS) + 읽기 속도 조절
    └─ 적응형 난이도 (Krashen i+1 기반 실시간 조정)

Layer 3. 서비스 전달
├── PWA 웹앱 (큰 글씨, UDL, 다국어 UI)
├── 카카오톡/문자 알림
└── 교사 대시보드 + 교육청 대시보드
```

### ⭐ 모듈 6 상세 — 자기주도 학습 지원 엔진

에누마 3원칙을 "학교 정보 이해" 영역에 최초로 적용한 모듈.

**기능 1: 쉬운 한국어 변환기**
- 입력: NEIS 공지 원문
- 처리: Gemini + 한국어 쉬운 글 프롬프트("초등 3학년 수준 어휘 + 한 문장 15자 이내 + 능동태")
- 출력: 쉬운 한국어 + 모국어 병기
- 근거: 에누마 토도한글 "자모음 절식 학습법"에서 영감 받은 **문장 단위 세분화**

**기능 2: 학습 카드 생성기 (에누마 글방 7단계 이식)**
- 알:음이 전달한 공지의 **핵심 교육 용어**를 자동 추출 → 에누마 글방 방식의 학습 카드로 변환
- 7단계: 글자 이름 → 음운 인지 → 자모 조합 → 자모 쓰기 → 어휘 → 쓰기 → 읽기 이해
- 예시: "수행평가" 카드 →
  - 글자 인식 (수/행/평/가)
  - 발음 (su-haeng-pyeong-ga)
  - 의미 (모국어 정의)
  - 문맥 속 예문
  - 퀴즈 (핸드트래킹 O/X)

**기능 3: 게이미피케이션 엔진**
- 학부모 연속 알림 확인 스트릭 → 배지
- 학생 주간 학습 용어 수 → 레벨
- 에누마 탐험 모드를 참고한 **"학교 정보 탐험"** UI (가정통신문 → 학사일정 → 급식 → 수행평가 → 진로 순으로 맵 해금)

**기능 4: 음성 + 속도 조절 (UDL)**
- 모든 텍스트에 TTS 버튼
- 읽기 속도 0.5x ~ 2.0x
- 학부모가 글을 못 읽는 경우에도 서비스 이용 가능 (에누마 XPRIZE 환경과 동일)
